# QUCC Hub プライバシーポリシー

## 1. 取得する情報
本アプリは、Strava APIを通じて以下の情報を取得します。
- ユーザーのアクティビティデータ（走行距離、獲得標高、開始日時、ルートのポリライン）
- Stravaの表示名など、連携に必要な基本プロフィール情報
- 本アプリ内で入力した入学年度、プロフィール文、自転車情報

## 2. 利用目的
取得した情報は、以下の目的のみに使用します。
- 九州大学サイクリング同好会（QUCC）内での活動記録の可視化
- 部員間のモチベーション向上およびルート共有

## 3. データの保護と共有
- 取得したデータは、本アプリのマップ上およびランキング表示にのみ使用し、第三者に販売または提供することはありません。
- 承認済みの QUCC メンバーおよび運営管理者は、アプリ内で共有対象となったルートと集計値を閲覧できます。
- Strava 連携の解除は `https://www.strava.com/settings/apps` から行えます。
- 同期済みデータの削除を希望する場合は `qucc.cycling@gmail.com` までご連絡ください。

## 4. Strava による情報取得について
- Strava は、Strava API の利用状況に関するデータを収集・利用する場合があります。
- 詳細は Strava の API Agreement および Privacy Policy をご確認ください。
