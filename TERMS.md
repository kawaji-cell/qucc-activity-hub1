# QUCC Hub 利用規約

## 1. 利用資格
本アプリは、九州大学サイクリング同好会（QUCC）の現役部員およびOB・OGを対象としています。
- 運営管理者は、登録内容を確認したうえで表示権限を承認または停止できます。

## 2. データ共有への同意
- Strava を連携すると、承認済み QUCC メンバーがアプリ内であなたの共有対象ルートとライド集計を閲覧できるようになります。
- 接続解除は `https://www.strava.com/settings/apps` から行えます。
- 同期済みデータの削除を希望する場合は `qucc.cycling@gmail.com` までご連絡ください。

## 3. 禁止事項
- 本アプリのデータを不正に抽出、改ざんすること。
- 他の部員のプライバシーを侵害する行為。

## 4. 免責事項
- 本アプリの利用により生じたトラブルや事故について、開発者は一切の責任を負いません。安全に配慮して走行してください。
